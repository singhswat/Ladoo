What you get as a partner

GiftBox sends you pre-validated gifting requests with customer consent.

  • ✔ Ready-to-transfer demat instructions
  • ✔ Parent KYC already completed
  • ✔ Auto-routed allocations for SIP, baskets, or stock gifting
  • ✔ Higher AUM inflow from family & gifting channels
  • ✔ API gateway or dashboard-based execution options

Technical Integration Overview

We support two integration models depending on your infrastructure.

1. API Integration (Recommended)

Your system receives a signed GiftBox instruction via HTTPS POST. Includes:
- Recipient demat
- Transfer type (stock, basket, amount, SIP)
- Parent details & consent
- ISIN-level breakdown

2. Dashboard Mode

Brokers can alternatively log into a secure GiftBox dashboard to view pending transfers and mark them as executed.
